# Sound Files

Place the following MP3 files in this directory:

- `goal.mp3` - Played when a goal is scored (e.g., crowd cheer)
- `whistle.mp3` - Played when timer starts/stops (e.g., referee whistle)
- `card.mp3` - Played when a card is given (e.g., short alert sound)

## Recommended Sources

Free sound effects can be found at:
- https://freesound.org
- https://mixkit.co/free-sound-effects/
- https://www.zapsplat.com

## File Format

- Format: MP3
- Duration: 1-3 seconds recommended
- Size: Keep under 100KB for fast loading
